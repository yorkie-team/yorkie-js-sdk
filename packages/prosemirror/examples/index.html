@yorkie-js/prosemirror

Collaborative editing examples. Requires a Yorkie server (docker compose -f docker/docker-compose.yml up -d).

  • Basic Example prosemirror-schema-basic with strong, em, code, link marks. Full toolbar via prosemirror-example-setup.
  • Custom Schema Example notegroup > note nesting, star inline node, boring_paragraph, shouting mark. Tests buildMarkMapping().
